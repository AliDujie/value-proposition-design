# Web Persona 人物角色创建与应用专家 Skill

基于《赢在用户：Web人物角色创建和应用实践指南》(The User Is Always Right, Steve Mulder & Ziv Yaar) 全书知识体系构建的完整 AI Skill。

## 📖 概述

本 Skill 将书中 12 章的完整知识体系沉淀为一个可随时调用的 AI 能力模块，覆盖人物角色(Personas)从创建到应用的全生命周期，既能作为教练指导团队，也能作为执行者独立完成每个环节。

## 🏗️ Skill 架构（10 大模块）

| 模块 | 对应章节 | 核心能力 |
|------|---------|---------|
| A. 创建方法选择 | 第3章 | 定性/经定量验证/定量 三种路径 |
| B. 定性用户研究 | 第4章 | 访谈/现场调查/可用性测试 |
| C. 定量用户研究 | 第5章 | 问卷/流量分析/CRM数据 |
| D. 用户细分 | 第6章 | 定性细分/定量聚类/验证 |
| E. 角色真实化 | 第7章 | 命名/照片/简介/语录/优先级/场景 |
| F. 保持活力 | 第8章 | 文档/展示/日常维护 |
| G. 商业策略 | 第9章 | 策略架构/跨渠道应用 |
| H. 功能定义 | 第10章 | 头脑风暴/优先级矩阵/竞品分析 |
| I. 结构内容设计 | 第11章 | IA/内容策略/视觉设计 |
| J. 成果衡量 | 第12章 | 发布前后测试/定量衡量体系 |

## 🎯 能力覆盖

- **12 项能力域**：研究规划、定性采集、定量采集、用户细分、角色创建、角色推广、商业策略、功能定义、结构设计、内容策略、视觉设计、成果衡量
- **双层能力**：教练级（指导团队）+ 执行者级（独立完成）
- **4 个实战工具模板**：研究计划、角色文档、功能矩阵、竞品分析
- **7 个经典案例**：VistaPrint、BrownCo、CNN.com、Best Buy、Philips、FedEx、Sony
- **12 项质量检查清单**
